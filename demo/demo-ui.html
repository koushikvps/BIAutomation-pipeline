ADO Connection Required

Go to Settings > Connectors to connect your Azure DevOps project. The pipeline needs this to fetch stories.

How it works — Commander + Supervisor Architecture

Paste an ADO Work Item ID (or type a free-text story). The Commander Agent analyzes it and creates a dynamic execution plan, dispatching worker agents (Planner, Developer, Validator) as needed. A Supervisor Agent independently validates every critical step and performs final sign-off. You approve the plan before anything touches your database.
